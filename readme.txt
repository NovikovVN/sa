0.1. На компьютере, где будет работать сервис, должены быть установлены все использующиеся в нем python-модули (flask, sclearn, pickle) желательно последних версий. Разработка велась на Python 3.6.4.
0.2. Подготовка и проверка модели представлены в файлах _preparation.py и _test.py соответственно.
0.3. Скрипт проверки _test.py и модуль с классом, реализующим модель, основаны на аналогах из демо-примера. Файл sentiment_demo.py и шаблон add_review.html основаны также на аналогах из демо-примера и на примере из видео-урока по flask https://www.youtube.com/watch?v=uyjp8nLGkmM
В шаблон добавлена bootstrap-стилизация.
1.1. Для запуска сервиса в консоли cmd установить путь к папке с сервисом и запустить файл sentiment_demo.py (как на скриншоте screenshots/01 run sentiment_demo.png).
1.2. Если запуск сервиса прошёл штатно, в консоли появится сообщение, что сервис работает по адресу http://127.0.0.1:5000.
1.3. Вид штатно запущенного сервиса и пример его работы представлены на скриншотах.
1.4. Прекратить работу сервиса можно в консоли cmd комбинациями Ctrl+C или Ctrl+Break.
